# SeaShell
Basic bash written in C++.

Uses WSL 

Runs ```myscript.txt``` at startup

#### Install readline
```sudo apt-get install libreadline-dev```

#### Run with readline
```g++ SeaShell.cpp -o a1 -lreadline && ./a1```
